# Library Management System

## Overview

The Library Management System is a simple Python-based project that helps users manage book records efficiently. This project allows users to add books, view available books, search for books using their ID, and delete books from the library through a menu-driven interface.

It is a beginner-friendly project that demonstrates the use of Python dictionaries, loops, conditional statements, and user input handling.

---

## Features

- Add New Books
- View All Books
- Search Book by ID
- Delete Book Records
- Menu-Driven Interface
- Easy to Use
- Exit System

## Working of the Project

### Add Book
Users can add one or more books by entering:
- Book Name
- Book ID

### View Books
Displays all books available in the library.

### Search Book
Users can search for a specific book using its Book ID.

### Delete Book
Users can delete a book record from the library.

### Exit
Terminates the program safely.
